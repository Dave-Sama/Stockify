# Stock Plotter
A modular stock charting app with Flask (API) and React (frontend).

## Setup
1. **Backend**:
   - `cd backend`
   - `pip install -r requirements.txt`
   - `python run.py`
2. **Frontend**:
   - `cd frontend`
   - `npm install`
   - `npm start`
3. Open `http://localhost:3000`.

## Structure
- `backend/`: Flask API serving Plotly JSON.
- `frontend/`: React app with form and chart components.
